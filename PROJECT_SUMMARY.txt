KEY IMPROVEMENTS: NETWORK FUNCTIONS
================================================================================

PROBLEM: Old functions didn't work on Windows 11
-------------------------------------------------

OLD CODE (Didn't Work):
    Get-NetAdapter | Where-Object {$_.Name -match "Wi-Fi|Wireless"}
    netsh interface set interface name="Ethernet" admin=disable
    # Failed because hardcoded names don't exist on all systems

NEW CODE (Works Everywhere):
    Get-NetAdapter -Physical | Where-Object {
        $_.MediaType -like "*802.11*" -or              # Detects Wi-Fi standard
        $_.InterfaceDescription -like "*wireless*" -or # Checks description
        $_.InterfaceDescription -like "*wi-fi*" -or
        $_.InterfaceDescription -like "*bluetooth*" -or
        $_.Name -like "*wi-fi*" -or                    # Checks name
        $_.Name -like "*wireless*"
    }
    # Works with ANY adapter name, multiple detection methods

RESULT: Network functions now work reliably on Windows 11!

HOW TO USE
================================================================================

METHOD 1: Interactive Menu (Recommended)
-----------------------------------------
1. Right-click: launchers\Launch-SystemControl.bat
2. Select: "Run as administrator"
3. Choose options (single or comma-separated)

Example:
    Selection: 8       (Disable wireless)
    Selection: 8,16    (Disable wireless + empty recycle bin)


METHOD 2: Quick Actions (Command Line)
---------------------------------------
PowerShell as Administrator:
    .\scripts\QuickActions.ps1 -Action DisableWifi
    .\scripts\QuickActions.ps1 -Action EnableAllNetwork
    .\scripts\QuickActions.ps1 -Action ListNetworkAdapters


METHOD 3: Standalone Scripts
-----------------------------
Double-click batch files in launchers\ folder
    - ImmediateRestart.bat
    - DelayedRestart.bat
    - Hibernate.bat

Or run PowerShell scripts:
    .\scripts\KeepAlive.ps1


METHOD 4: Import as PowerShell Module
--------------------------------------
Import-Module .\modules\SystemControlFunctions.psm1
Disable-WirelessAdapters
Enable-AllNetworkAdapters

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Access Denied"
SOLUTION: Right-click → Run as administrator

PROBLEM: "Execution policy" error
SOLUTION: Use the batch launchers (they handle this automatically)

PROBLEM: Network function doesn't work
SOLUTION: Check adapter names with:
    Get-NetAdapter -Physical | Format-Table Name, Status, MediaType

PROBLEM: "Module not found"
SOLUTION: Ensure you're in the correct directory or use full path:
    Import-Module "C:\...\modules\SystemControlFunctions.psm1"
